介 绍

     新西城区历史底蕴丰厚，3000多年的建城史从这里开始，800多年的建都史从这里溯源。新西城是北京文物古迹大区。文物古迹是历史文化的重要载体，凝聚着深厚而丰富的文化内涵，它们像一部部史书、一卷卷档案，记录着城市的沧桑岁月，保护好这些珍贵的文物古迹，会使城市的历史延绵不绝，会使城市永远焕发着悠久的魅力和时代的光彩。为了更全面的向公众普及新西城的文物古迹，出于公益目的我们制作了本手册。本手册收录了北京市西城区范围内的国、市、区三级文物保护单位图文资料，包括相关的历史文化保护区及街道办事处范围，因收录内容较多，加之水平有限，难免有错误和疏漏之处，敬请谅解，恳请指正。
